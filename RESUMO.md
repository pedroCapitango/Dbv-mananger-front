## 🚀 Como Começar

### Setup Inicial (1 minuto)
```bash
# 1. Instalar dependências
pnpm install

# 2. Configurar ambiente
cp .env.example .env

# 3. Iniciar servidor
pnpm dev
```

### Acessar Sistema
```
http://localhost:5173
```

### Login de Teste
- Email: qualquer@email.com
- Senha: qualquersenha
