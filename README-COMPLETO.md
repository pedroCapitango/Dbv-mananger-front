## 🚀 Como Usar

### 1. Instalar Dependências
```bash
pnpm install
```

### 2. Configurar API
```bash
# Arquivo .env (opcional)
VITE_API_BASE_URL=https://clube-black-api.onrender.com
```

### 3. Iniciar Servidor
```bash
pnpm run dev
```

Acesse: **http://localhost:5175**
